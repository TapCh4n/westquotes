## 🇺🇸

WestQuotes is an Android application with widgets featuring daily quotes from Kanye West.

### Features
- Daily Quote: A new quote delivered every day.
- Custom Widgets: Highly customizable home screen widgets.
- Favorites: Save your favorite quotes to a dedicated list.
- Themes: Multiple app themes to choose from.

### Installation
1. Download APK: Go to the Releases section and download WestQuotes.apk.
2. Allow Installation: If prompted, allow your browser to install apps from unknown sources.
3. Play Protect: If an Unknown Developer warning appears, click More details and then Install anyway.

## 🇵🇱

WestQuotes to aplikacja na Androida z codziennymi cytatami Kanyego Westa.

### Funkcje
- Codzienny cytat: Nowy cytat dostarczany każdego dnia.
- Własne widżety: Możliwość personalizacji widżetów na ekranie głównym.
- Ulubione: Opcja dodawania wybranych cytatów do listy ulubionych.
- Motywy: Wiele motywów wizualnych aplikacji do wyboru.

### Instalacja
1. Pobierz plik APK: Przejdź do sekcji Releases i pobierz WestQuotes.apk.
2. Zezwól na instalację: Jeśli system zapyta, zezwól przeglądarce na instalację z nieznanych źródeł.
3. Play Protect: Jeśli zobaczysz komunikat Nieznany twórca, wybierz Więcej szczegółów i kliknij Zainstaluj mimo to.
